# hydraidectl User Manual

> **The manual moved.** Each command now has a focused page under
> [`README.md`](README.md), grouped by category. This single-file manual
> was retired because it had grown to ~1300 lines and was hard to
> navigate.

Pick your destination:

- 📖 [**Index**](README.md) — full command map, conventions, privilege rules.
- 🚀 [**Instance lifecycle**](lifecycle.md) — `init`, `edit`, `start`, `stop`, `restart`, `destroy`.
- 📊 [**Monitoring & status**](monitoring.md) — `list`, `health`, `observe`, `explore`, `telemetry`, `version`.
- ⬆️ [**Upgrades & migration**](upgrades.md) — `upgrade`, `migrate v1-to-v2`, `migrate v2-migrate-format`, `engine`.
- 💾 [**Data management**](data.md) — `backup`, `restore`, `compact`, `cleanup`, `size`, `stats`, `inspect`.

If you're brand new, the two-command quickstart is at
[`docs/install/quickstart.md`](../install/quickstart.md).
